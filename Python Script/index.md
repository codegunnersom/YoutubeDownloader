 # AI Youtube Clip Downloader.

### Vision / Use Cases

#### 1. The first part is to just download youtube videos from the internet. 
#### 2. Will try to download the only clipped instance of the video. 
#### 3. Will also download Full video in a temp location, just in case few videos need to be downloaded locally. 
#### 4. Will Also integrate Audio Download Support. 


### Future Development plan : 

#### Need to Integrate Chat-GPT support, so that AI can be prompted for the use-case or requirement of clip. AI then will provide the link of the video/Audio with the specific Time-Stamps.And then the basic/ Primary application can be used to download the clip with other functionality offered by the base app.
